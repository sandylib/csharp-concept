Class customer is the Type and fields,properties and method are type members.

In general classes, structs,enums,interfaces,delegates are called as types and
fields,properties,constructors,methods etc,that normarlly reside in a type 
are called as type members.

In C# there are 5 different access modifiers:
1.Private
2.Protected
3.Internal
4.Protected Internal
5.Public


Type members can have all the access modifiers, where as types can have only
2(internal,public)of the 5 access modifiers
